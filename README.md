# Kina Mono-Repo Scaffold

This repository contains a production-grade mono-repo skeleton without business logic.

## Getting started

1. Create a local environment file:

```bash
cp .env.example .env
```

2. Build and run the stack:

```bash
docker compose up --build
```

## Available URLs

- http://localhost/ (webapp)
- http://localhost/admin/ (admin)
- http://localhost/api/health (api health)
